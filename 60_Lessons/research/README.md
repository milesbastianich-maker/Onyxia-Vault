# Research Logs

Output target for `onyxia-researcher` cross-vault sweeps and pattern hunts that do not map to a single account, contact, competitor, vertical, or regulatory file.

## File naming

`YYYY-MM-DD-{topic-slug}.md`

## When to write here vs. elsewhere

| Task | Target |
|---|---|
| Account deep-dive | `10_Accounts/{Company}.md` `## Research notes` block |
| Contact deep-dive | `20_Contacts/{First Last}.md` `## Research notes` block |
| Single competitor | `50_Intel/competitors/{tool}.md` |
| Single regulation | `50_Intel/industry/regulatory-calendar.md` |
| Single vertical | `50_Intel/verticals/{vertical}.md` |
| Cross-vault sweep / pattern hunt that touches multiple of the above | here |
| Single trigger event scan | `50_Intel/signal-log/YYYY-MM-DD-{topic}.md` |
