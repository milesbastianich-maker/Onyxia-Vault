## Positioning frame

Onyxia sits above the security stack. Per [[../product/positioning]]: "We sit above your existing security stack, normalize the data, and auto-generate board-ready reporting." That is the layer of differentiation. If a prospect has one of these tools, Onyxia does not replace it. It reads from it.

## How to read these files

- "Where Onyxia wins" only cites patterns we can actually defend from [[../product/features]], [[../product/quotes]], [[../playbook/closed-won-patterns]].

- If the vault is silent on a differentiation dimension, the file says so. Do not invent.
