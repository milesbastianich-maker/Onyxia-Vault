## Regulatory surface

- **CMMC 2.0 Level 2 Certification — November 10, 2026 (hard deadline).** Contracting officers will begin requiring C3PAO-assessed Level 2 certification as the default for DoD contracts involving CUI. No grace period. No extension announced. Source: [RidgeIT CMMC Phase 2 Guide](https://www.ridgeit.com/cmmc-phase-2-deadline-november-2026/).
- **Scale:** ~220,000 DIB contractors affected. C3PAO assessor slots booking into late 2026 and 2027 already. Only ~1% of affected contractors are fully prepared.
- **Readiness timeline:** 12-14 months minimum. Gap analysis → remediation → documentation → pre-assessment review → C3PAO engagement. Any contractor starting today (April 2026) is already behind the 12-month curve.
- **NIST 800-171 Rev 3 (underlying framework).** DoD April 2025 memo assigned ODP values for Rev 3 as the minimum standard for contractors. CMMC Level 2 is built on NIST 800-171 controls — the same framework higher-education DoD research contracts use.
- **DFARS clause 252.204-7012.** Already required adequate security on covered defense systems. CMMC formalizes this with independent assessment. Non-compliance = loss of contract eligibility.
- Source: [Secureframe CMMC 2.0 Timeline](https://secureframe.com/hub/cmmc/proposed-final-rule), [CRC Cloud CMMC Countdown](https://www.crccloud.com/cmmc-2-0-compliance-countdown-what-small-defense-contractors-need-to-do-before-november-2026).

## What defense / aerospace CISOs care about

1. **CMMC certification evidence.** CMMC is a documentation audit as much as a technical one. C3PAO assessors review whether controls are documented, continuous, and verifiable. Onyxia's continuous monitoring layer IS that documentation artifact.
2. **November deadline pressure.** Any CISO at a DIB company that hasn't started is in crisis mode. The urgency hook needs zero education — it sells itself.
3. **Multi-program CUI scoping.** Large defense contractors have multiple programs with different classification levels. A single-pane view of which programs are CMMC-ready matters.
4. **Supply-chain risk (CMMC flows down).** Prime contractors must ensure their subcontractors are also CMMC-certified. Onyxia's third-party monitoring layer applies to the supply-chain documentation requirement.
