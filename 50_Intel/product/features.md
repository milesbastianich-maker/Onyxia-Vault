# Features

## Core architecture

**Data fabric.** Collects from the entire security ecosystem via API. Cleans, maps, correlates, deduplicates, normalizes.

**Integration time.** "Connect your entire security stack in minutes." Premera POC doc confirms 2-4 week POC duration for integrating X technologies.

## The eight feature pillars

### 1. CPI Library (Cybersecurity Performance Indicators)

Pre-built library of CPIs aligned with multiple frameworks. Customers select which CPIs to measure, connect multiple resources per CPI, see the exact formula, and adjust SLA to program needs.

CPI domains:
- Detection & Response
- Vulnerability Management
- Training & Awareness
- Cloud Security
- Identity & Access Management
- Device Management

Custom labels allow adding text to any CPI chart (e.g., "headcount change," "new process," "new product"). Historical CPI data over the past year.

### 2. Security Stack Map

Auto-generates on integration. Each integrated product gets charted into 1 of 30 cells based on:
- Asset type (Devices, Applications, Networks, Data, Users)
- NIST 2.0 pillar (Identify, Protect, Detect, Respond, Recover, Govern)

Value: identify gaps + overlaps, optimize stack efficiency, yield cost savings, align with framework.

### 3. Asset Coverage Analyzer

Measure and compare security tool coverage across all devices. Ensure no gaps or overlaps on specific assets.

### 4. Automated Board & Executive Reporting

- Full report on all CPIs (one click)
- Focused report (select CPIs)
- Monthly preset
- Quarterly preset
- PDF downloadable

Each report includes: table of contents, granular CPI look, contextualized performance labels, definition + business value of each metric, performance over time.

### 5. Benchmarking

Compare program performance against average SLA in different industries. Allows comparison to more highly regulated industries so customers can raise their own posture accordingly. On-demand, historical, real-time.

### 6. Budget View

Maximize security program spend with automated measuring. Prioritize investments based on actionable intelligence. Clear picture of budget allocation. Measure ROI based on product performance and utilization.

### 7. Framework Compliance Dashboard

One-click transform of the entire dashboard to align with a chosen framework:
- NIST CSF 2.0 (with new "Govern" function)
- HIPAA
- HITRUST
- FedRamp
- SOC 2
- PCI
- Custom (ISO)

Parent security categories + CPIs automatically remap.

### 8. OnyxAI (Predictive Cybersecurity Management Engine)

Statistical analysis + machine learning + NLP models applied to the organization's entire security ecosystem.

Delivers:
- Suggestions to improve program performance
- SLA optimization
- Program trend predictions
- Risk reduction recommendations
- Crisis prevention via forecast

### 9. Nexa (AI Security Agent)

Moves beyond generic AI. Analyzes security data + asset inventories in the specific context of the organization's unique environment.

Capabilities:
- Answer questions across all security domains instantly
- Eliminate manual correlation
- Trustworthy, actionable insights
- Recommended actions for preemptive exposure mitigation
- Integrates directly into the organization's cybersecurity ecosystem

Example Nexa queries (from Brad Baldelli's training doc):
- Identity: "How many users logged into the network since last week?" / "Show me 10 users that logged in recently" / "Tell me more about user X"
- Email: "How many suspicious emails have we received this past year?" / "Show me a breakdown of emails per month"

### 10. Mobile App

Security program at CISO fingertips anytime, anywhere.
