# Hunt Mode

Proactive net-new customer acquisition. Four hunt patterns, each tuned for a different signal source.

## The four hunts

| Hunt | File | Cadence | Signal Source | Weekly Volume Target |
|---|---|---|---|---|
| Trigger-event hunt | [[trigger-event-hunt]] | Daily AM | ZoomInfo Scoops, web news, SEC EDGAR | 3-5 fresh triggers/day |
| ICP lookalike hunt | [[icp-lookalike-hunt]] | Mondays | Closed-won corpus → ZoomInfo similar companies | 10-15 net-new/week |
| Vertical saturation hunt | [[vertical-saturation-hunt]] | Rolling (1 vertical/month) | ZoomInfo + industry lists | Full vertical coverage |
| Intent signal hunt | [[intent-signal-hunt]] | Continuous (via RB2B) | RB2B + ZoomInfo intent + form fills | All qualified visitors |

## The hard rule for all hunts

Every hunt feeds through:
1. `50_Intel/icp.md` fit scoring
2. Exclusion list check
3. HubSpot dedupe (is this already owned by another rep?)
4. Vault dedupe (is this already in `10_Accounts/`?)

No hunt output goes to Miles without passing all four.

## Target combined output

40-50 qualified Tier 1 prospects staged per week. Mapped to the autopilot SPEC goal of 40-50 hyper-personalized sends/day.
